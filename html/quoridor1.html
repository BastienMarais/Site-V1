Présentation du projet :

Ce projet a été réalisé lors de l'épreuve d'ISN pour le Bac. Le Quoridor consiste à déplacer deux pions sur un quadrillage afin d'arriver sur le côté opposé. Cependant, le joueur a le choix à chaque tour de se déplacer ou de poser un mur. Dans cette version, le jeu est fonctionnel mais les murs sont traversable.
Nous l'avons réalisé à quatre en un mois.
