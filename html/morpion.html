Présentation du projet :

Le Morpion a été réalisé dans le but de tester nos toutes nouvelles connaissances en C lors de la terminale. C'est la version numérique du jeu "Morpion" qui consiste à aligner 3 pions pour gagner. Cette version du projet permet de faire s'affronter deux joueurs sur le même écran.
Je l'ai réalisé seul lorsque ma connaissance du langage C était encore très limitée.
