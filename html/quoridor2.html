Présentation du projet :

Ce projet a été créé de manière autonome afin d'améliorer la version 1.0 qui avait été conçu avec de nombreux manques de connaissance. Le Quoridor consiste à déplacer deux pions sur un quadrillage afin d'arriver sur le côté opposé. Cependant, le joueur a le choix à chaque tour de se déplacer ou de poser un mur. Dans l'état actuel, le programme permet seulement d'afficher les différentes interfaces et de dessiner le quadrillage. Cependant le programme est toujours en cours de développement.
Pour réaliser ce projet je travaille sur mon temps libre.
